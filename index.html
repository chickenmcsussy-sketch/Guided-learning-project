About The Project

This project explores how modern computers work internally, from the CPU and memory hierarchy to motherboards, storage devices and hardware communication. Through interviews, research and practical disassembly, I aim to bridge theoretical knowledge with real-world experience.

Industry Interviews

To gain practical insights beyond textbooks, I interviewed professionals with different backgrounds in computing and hardware.

Dad

Network Security
  1. How did you first get into IT?
  2. How important is hardware knowledge when working with firewalls?
  3. What hardware failures have you encountered most?
  4. How do you usually troubleshoot systems?
  5. How has computer hardware changed over time?
  6. What advice would you give future engineers?

Hardware Specialist

Computer Hardware
  1. How did you become interested in hardware?
  2. How would you build a PC from scratch?
  3. What is the most overlooked component?
  4. How would you explain CPU, RAM and motherboard?
  5. What misconceptions do students usually have?
  6. Which resources helped you learn hardware?
