Hardware Explorer

Click each component to explore its purpose and what I learnt.

Central Processing Unit

The CPU executes instructions and controls every operation performed by the computer.

Random Access Memory

Stores temporary data needed by programs currently running.

Motherboard

Connects every component and enables communication between hardware.

Storage

Stores the operating system, applications and personal files.

Graphics Card

Processes graphics and accelerates visual rendering.

Power Supply

Converts AC power into stable DC voltage for every component.
